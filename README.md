## Описание проекта

Данный репозиторий содержит рабочий прототип рекомендательной системы, реализующей два подхода:
- **Математическое моделирование** — матричная факторизация ALS
- **Искусственный интеллект** — нейронная сеть с BPR-функцией потерь

Проект разработан в рамках НИР: **«Применение математического моделирования и искусственного интеллекта для разработки рекомендательных систем»**.

### Используемый датасет

**T-ECD (T-Bank E-Commerce Cross-Domain Dataset), домен Marketplace**

| Параметр | Значение |
| :--- | :--- |
| Событий всего | 130 447 117 |
| Пользователей (после фильтрации) | 1 382 085 |
| Товаров (после фильтрации) | 760 794 |
| Взаимодействий в матрице | 127 067 374 |
| Плотность матрицы | 0,0079% |

## Научная новизна

1. **Первое применение ALS и Neural BPR к датасету T-ECD Marketplace** — ранее в открытых публикациях не встречалось сравнительного анализа этих методов на данном датасете.

2. **Адаптация под ограниченные ресурсы** — модели обучены в среде Google Colab на CPU с подвыборкой данных.

3. **Количественная оценка пределов персонализации** — экспериментально подтверждено, что при плотности матрицы 0,008% персонализированные модели не превосходят простой baseline.

### Выводы

Эксперимент показал, что в условиях экстремальной разреженности данных (ρ = 0.008%) персонализированные коллаборативные модели (ALS, Neural BPR) не способны превзойти простой не-персонализированный подход (Popular). Это указывает на необходимость использования гибридных моделей с контентными признаками.
