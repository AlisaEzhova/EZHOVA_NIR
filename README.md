## Описание проекта

Данный репозиторий содержит рабочий прототип рекомендательной системы на основе **матричной факторизации** с использованием библиотеки PyTorch. Проект разработан в рамках НИР: **«Применение математического моделирования и искусственного интеллекта для разработки рекомендательных систем»**.

### Цель исследования
Разработка и апробация математических моделей и методов искусственного интеллекта для повышения точности персонализации рекомендательных систем на основе анализа пользовательских предпочтений и контента.

### Ключевая идея
Использование нейросетевой матричной факторизации (Embedding + bias-термы) для предсказания пользовательских оценок и генерации top-N рекомендаций с возможностью дальнейшей интеграции с LLM для решения проблемы «холодного старта».

---

## Научная новизна

1. **Гибридный потенциал**: В отличие от классической матричной факторизации, реализованная модель на PyTorch допускает расширение до нейроколлаборативной фильтрации (NCF) и интеграции с LLM.

2. **Производственно-готовый стек**: Использование FastAPI + PostgreSQL обеспечивает возможность развертывания в production-среде.

3. **Сравнительный анализ**: Проведено сравнение с базовыми подходами (рекомендация популярного, случайные рекомендации) по метрикам Precision, Recall, RMSE.

## Стек технологий

| Компонент | Технология | Обоснование выбора |
|:---|:---|:---|
| **Язык** | Python 3.10+ | Стандарт для ML и анализа данных |

### Почему PyTorch?
- Динамическое построение вычислительных графов (удобно для исследований)
- Широкое применение в рекомендательных системах
- Легкая интеграция с FastAPI для инференса

## Установка и запуск

### Вариант 1: Локальный запуск (рекомендуется)

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/ВАШ_АККАУНТ/recommender-proto.git
cd recommender-proto

# 2. Создайте виртуальное окружение
python -m venv venv

# 3. Активируйте окружение (Windows)
venv\Scripts\activate

# 4. Установите зависимости
pip install -r requirements.txt

# 5. Скачайте данные MovieLens
# Перейдите на https://grouplens.org/datasets/movielens/latest/
# Скачайте ml-latest-small.zip и распакуйте ratings.csv в папку data/

# 6. Обучите модель
py -m app.train

# 7. Запустите API сервер
py -m uvicorn app.main:app --reload

# 8. Откройте в браузере
# http://localhost:8000/docs
